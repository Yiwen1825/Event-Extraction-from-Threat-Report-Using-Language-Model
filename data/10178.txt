 Cisco releases fixes for remote code execution flaws in Webex Network Recording Player 
Cisco has patched a set of severe vulnerabilities which could lead to remote code execution in the Cisco Webex Network Recording Player for Advanced Recording Format (ARF). | PatchVulnerability
The security flaws, CVE-2018-15414, CVE-2018-15421, and CVE-2018-15422, have been issued a base score of 7.8. | DiscoverVulnerability
According to the Cisco Product Security Incident Response Team (PSIRT), the flaws could lead to "an unauthenticated, remote attacker to execute arbitrary code on a targeted system." | O
The Cisco Webex Network Recording Player for Advanced Recording Format (ARF), available for Windows, Mac, and Linux machines is a component for recording meetings taking place in the Cisco Webex Meetings Suite sites, Cisco Webex Meetings Online sites, and Cisco Webex Meetings Server. | O
In a security advisory posted this week, Cisco says that the following software is affected:  | O
    Cisco Webex Meetings Suite (WBS32): Webex Network Recording Player versions prior to WBS32.15.10;  | O
    Cisco Webex Meetings Suite (WBS33): Webex Network Recording Player versions prior to WBS33.3;  | O
    Cisco Webex Meetings Online: Webex Network Recording Player versions prior to 1.3.37;  | O
    Cisco Webex Meetings Server: Webex Network Recording Player versions prior to 3.0MR2. | O
According to Cisco, each operating system is vulnerable to at least one of the security flaws. | DiscoverVulnerability
The vulnerabilities are due to the improper invalidation of Webex recording files. | O
 If a victim opens a crafted, malicious file in the Cisco Webex Player -- potentially sent over email as part of a spear phishing campaign -- the bugs are triggered, leading to exploit. | AttackPhishing
